Keyboard Shortcuts

  • ↑ / ↓ — Navigate tasks
  • Shift + ↑ / ↓ — Reorder tasks
  • Ctrl + ↑ / ↓ — Switch categories
  • Space — Toggle complete for focused task
  • Enter — Edit focused task
  • Delete / Backspace — Delete focused task
  • N — Place cursor in new task input field
  • Shift + N — Place cursor in new category input
  • Shift + C — Clear all completed tasks
  • Esc — Cancel edit
