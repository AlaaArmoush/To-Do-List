Keyboard Shortcuts

  • I — Open keyboard shortcuts
  • N — Focus new task input
  • Shift + N — Focus new category input
  • ↑ / ↓ — Navigate between tasks
  • Space — Mark task as complete or incomplete
  • Enter — Edit selected task
  • Delete / Backspace — Remove selected task
  • Shift + ↑ / ↓ — Move task up or down
  • Ctrl + ↑ / ↓ — Switch between categories
  • Ctrl + Shift + ↑ / ↓ — Reorder category positions
  • Shift + C — Delete all completed tasks
  • Esc — Cancel current edit
